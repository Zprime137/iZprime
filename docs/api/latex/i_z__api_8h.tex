\doxysubsection{Detailed Description}
Public API for prime sieving, range scans, and prime generation. 

This header provides the stable, high-\/level entry points for the iZ library. Most functions return a heap-\/allocated {\ttfamily UI64\+\_\+\+ARRAY\texorpdfstring{$\ast$}{*}} that must be released with \doxylink{group__iz__arrays_gab1a0ced5e600649da6e519693a3a2851}{ui64\+\_\+free()}.
